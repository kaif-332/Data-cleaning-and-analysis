## 📌 Overview

This project analyzes **large-scale agricultural data across India**, focusing on **state-wise crop production and yield patterns**.

The goal is to extract insights about how factors like **crop type, rainfall, and cultivated area** influence overall productivity.

---

##  Dataset Features

The dataset includes:

*  **State** – Indian states
*  **Crop Type** – e.g., Rice, Wheat, Pulses
*  **Rainfall (mm)** – Annual/seasonal rainfall
*  **Production (tons)** – Total crop production
*  **Area (hectares)** – Cultivated land
*  **Average Yield (kg/hectare)** – Productivity metric

---

 Tech Stack

* **Python 3**
* **NumPy** – Data manipulation & numerical operations
* **Matplotlib** – Data visualization

---

 Workflow

 1. Data Cleaning

* Removed missing/null values
* Standardized units and formats

 2. Data Processing

* Computed average yield where needed
* Aggregated data by state and crop

 3. Analysis

* Compared production across states
* Studied rainfall vs yield relationships
* Identified high and low performing regions

---

 Visualizations

* 📊 **Bar Charts** – State-wise production comparison
* 📉 **Line Graphs** – Rainfall vs yield trends
* 🔵 **Scatter Plots** – Correlation analysis
* 🥧 **Pie Charts** – Crop distribution

---

 Key Insights

* 🌍 Certain states dominate specific crop production due to environmental advantages
* 🌧️ Rainfall has a **variable but significant impact** on yield
* ⚖️ Some regions show **high production but low efficiency**, indicating optimization potential
